SYSTEM DETAILS (for documentation generation)

Project Title:
Web-Based Academic Repository with Algorithmic Plagiarism Detection Using TF-IDF and Cosine Similarity

Project Type:
Web application for academic document storage, plagiarism checking, and report generation.

Target Users:
1. Guest User
- Can browse limited/public document information.
- Cannot access protected analysis actions.

2. Registered User (Student/Researcher)
- Can register/login.
- Can upload PDF, DOCX, TXT files.
- Can enter metadata (title, authors, course/program, year, abstract).
- Can run plagiarism analysis for own documents.
- Can view own complete reports.
- Can use quick scan (temporary analysis).

3. Admin User
- Can access admin dashboard.
- Can manage users and documents.
- Can view full reports across users for monitoring.

Core Features:
- JWT authentication and role-based access control.
- Document upload and metadata collection.
- TF-IDF + cosine similarity checking.
- Paragraph-level similarity highlighting.
- Similar title checking before upload.
- Optional AI semantic analysis (OpenAI).
- Internet source match checking.
- Quick Scan without permanent save.
- Owner-only report visibility (with admin override).

Technology Stack:
- Frontend: HTML, CSS, JavaScript (vanilla)
- Backend: Node.js, Express.js
- Database: Supabase PostgreSQL
- Security: JWT, bcryptjs, protected endpoints
- File processing: Multer, PDF parser, DOCX parser
- Deployment: Render + GitHub
